# 🌍 Streamlit Country Dashboard

A simple and interactive data visualization app built using **Streamlit**.  
This app allows users to explore country-level statistics like life expectancy and GDP using public data.

---

## 🚀 Features

- 📄 View dataset for selected country  
- 📈 Life Expectancy trend over time  
- 📊 Top 10 countries by GDP per capita  
- 🌍 Scatter plot (GDP vs Life Expectancy)  
- 🎛 Filters by continent and country  

---

## 📂 Dataset

This app uses the **Gapminder dataset**:
https://raw.githubusercontent.com/plotly/datasets/master/gapminderDataFiveYear.csv

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib
